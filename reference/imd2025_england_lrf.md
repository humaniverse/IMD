# Index of Multiple Deprivation (2025) for LRFs (2024) in England

A dataset containing IMD scores, ranks, and related measures for Local
Resilience Forums in England.

## Usage

``` r
imd2025_england_lrf
```

## Format

A data frame of class "tbl":

  Local Resilience Forum Code (2024)

- lrf24_name:

  Local Resilience Forum Name (2024)

  IMD - Average rank

  nationally
