# Index of Multiple Deprivation (multiple years) for LSOAs across the UK

A data set containing IMD deciles for the UK. Each nation deciled
separately.

## Usage

``` r
imd_uk_lsoa
```

## Format

A data frame of class "tbl":

- area_code:

  LSOA code or devolved equivalent

- IMD_decile:

  IMD decile

## Source

<https://www.gov.uk/>
